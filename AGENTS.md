# AGENTS.md - Moguri

Moguri is a local-first project repo for turning reference materials and recorded conversations into a member-shareable static site.

Before editing this repo:

1. Print `pwd`.
2. Run `git rev-parse --show-toplevel`.
3. Run `git status -sb`.
4. Read this `AGENTS.md`.

Repository rules:

- Do not commit raw audio or private source files by default.
- Keep publishable content in static GitHub Pages-friendly files at the repo root.
- Put source notes and transcripts under `docs/`.
- Put generated analysis data under `data/` and generated visual assets under `assets/`.
- Treat GitHub pushes, repo creation, and Pages publication as human-gated actions.
